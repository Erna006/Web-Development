My Goals for This Year

Posted on January 29, 2024

As I continue my journey in web development, I've set some goals for this year:

  1. Master HTML and CSS completely
  2. Learn JavaScript to make interactive websites
  3. Build at least 10 different projects
  4. Create my own portfolio website
  5. Learn about responsive design

I believe that consistent practice is the key to success. Every day I try to code for at least an hour, even if it's just small exercises or reading documentation.

The web development community is amazing - there are so many free resources like Khan Academy, freeCodeCamp, and MDN Web Docs. I'm grateful to have access to all this knowledge!
